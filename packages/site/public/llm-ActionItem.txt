# Action Item

Action items are used to display a single action that a user can take. They
are commonly used in lists to show a collection of actions.

## Design & usage guidelines

Action items are designed to be used in lists or collections where each item
represents a single action that can be taken by the user. They provide a
consistent way to display actionable content with optional icons, descriptions,
and interactive states.

### Basic usage

Use action items to create lists of actions or options that users can select
from:

```tsx
<ActionItem
  label="Edit profile"
  onClick={() => console.log('Edit profile clicked')}
/>
```

### With icons

Add icons to provide visual context for the action:

```tsx
<ActionItem
  label="Settings"
  icon="settings"
  onClick={() => console.log('Settings clicked')}
/>
```

### With descriptions

Include additional context with a description:

```tsx
<ActionItem
  label="Export data"
  description="Download your data as a CSV file"
  icon="download"
  onClick={() => console.log('Export clicked')}
/>
```

## Content guidelines

### Labels
- Use clear, action-oriented language
- Keep labels concise and descriptive
- Use sentence case for labels

### Descriptions
- Provide helpful context when needed
- Keep descriptions brief and informative
- Use sentence case for descriptions
