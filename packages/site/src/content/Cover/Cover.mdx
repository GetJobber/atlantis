# Cover

The Cover component helps you create full-height layouts with vertically
centered content. It's perfect for creating hero sections, login pages, or any
layout where you want content to fill the available vertical space.

## Design & usage guidelines

Use Cover when you need to:

- Create full-height layouts with centered content
- Build hero sections or splash screens
- Design login or registration pages
- Display content that should fill the available vertical space

## Basic example

```jsx
<Cover minHeight="100vh">
  <Cover.Center>
    <Heading>Welcome back!</Heading>
    <Text>Sign in to continue to your account.</Text>
    <Form>{/* Form content */}</Form>
  </Cover.Center>
</Cover>
```

## Props

### Cover

| Prop        | Type              | Description                                            | Default     |
| ----------- | ----------------- | ------------------------------------------------------ | ----------- |
| `children`  | `React.ReactNode` | The content to be displayed within the Cover component | Required    |
| `minHeight` | `string`          | Sets the minimum height of the Cover component         | `undefined` |
| `space`     | `string`          | Sets the spacing between child elements                | `undefined` |

### Cover.Center

| Prop       | Type              | Description                                                      | Default  |
| ---------- | ----------------- | ---------------------------------------------------------------- | -------- |
| `children` | `React.ReactNode` | The content to be vertically centered within the Cover component | Required |

## Subcomponents

### Cover.Center

Use `Cover.Center` to vertically center content within the Cover component.
Content within `Cover.Center` will be automatically centered in the available
space.

```jsx
<Cover minHeight="100vh">
  <Cover.Center>{/* Centered content */}</Cover.Center>
</Cover>
```

## Best practices

- Use meaningful minimum heights that make sense for your content
- Consider the viewport height (`vh`) units for full-screen covers
- Ensure content within `Cover.Center` is properly structured for vertical
  centering
- Use appropriate spacing values that match your design system
