# Stack

The Stack component creates a vertical layout of elements with consistent
spacing between them. It's one of the fundamental layout components in Atlantis
that helps create clean, organized interfaces.

## When to use

Use Stack when you need to:

- Create vertical layouts with consistent spacing
- Build forms with evenly spaced fields
- Layout content in cards or modals
- Create sections with equal vertical rhythm

## Props

### space

Controls the vertical spacing between elements. Accepts either a design token or
a custom value.

Available space tokens:

- `minuscule`
- `smallest`
- `smaller`
- `small`
- `base` (default)
- `large`
- `larger`
- `largest`
- `extravagant`

### recursive

When `true`, applies spacing to all nested elements. When `false` (default),
only applies spacing to direct children.

### splitAfter

Creates a split in the stack after the specified child index (1-15), pushing the
remaining content to the bottom. Useful for creating layouts with footer
content.

Relies on the height of the parent element to be larger than the height of the
stack.

## Examples

### Basic Stack

<Canvas>
  <Stack>
    <Text>First item</Text>
    <Text>Second item</Text>
    <Text>Third item</Text>
  </Stack>
</Canvas>

### Custom Spacing

<Canvas>
  <Stack space="large">
    <Text>More space</Text>
    <Text>Between items</Text>
  </Stack>
</Canvas>

### Split Layout

<Canvas>
  <div style={{ height: "200px" }}>
    <Stack splitAfter={2}>
      <Text>Main content</Text>
      <Text>Secondary content</Text>
      <Text>Footer content</Text>
    </Stack>
  </div>
</Canvas>

    </div>
    <div>
      <Text>Nested item 2.1</Text>

    </div>

## Accessibility

Stack uses semantic HTML and maintains proper spacing for readability. The
component preserves the natural document flow, ensuring good keyboard navigation
and screen reader compatibility.
