# UseCollectionQuery

`useCollectionQuery` is a generalized hook that that fetches a collection of
data. For use with any lists populated by graphQL collections.

```tsx
import { useCollectionQuery } from "@jobber/hooks";

const {
    data,
    error,
    refresh,
    nextPage,
    loadingRefresh,
    loadingNextPage,
    loadingInitialContent,
    } = useCollectionQuery<ListQueryType, SubscriptionType>({
        query: LIST_QUERY,
        queryOptions: {
            fetchPolicy: "network-only",
        },
        getCollectionByPath(items) {
            return items.path.toCollection;
        },
        subscription: {
            document: SUBSCRIPTION_QUERY,
            getNodeByPath(items) {
                return items.path.toNode
            },
    });
```
