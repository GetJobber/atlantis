# FormatFile

FormatFile allows users to view a preview of a file attachment.

```ts
import { FormatFile } from "atlantis/FormatFile";
```

## Usage guidelines

Use FormatFile for most instances where you need to display an image or other
file, particularly if the user can upload the file.
