# Foundation

Foundational colors and rhythm for the Jobber Atlantis Design System

## Installation

```sh
npm install @jobber/foundation
```

## Usage

`@jobber/foundation` ships `foundation.css`.

Import the `@jobber/foundation` stylesheet into your own css

```css
@import "~@jobber/foundation/foundation.css";

.myLabel {
  color: var(--color-blue);
}
```

Inject `@jobber/foundation` into your css with `postcss`

```js
module.exports = {
  plugins: [
    require("postcss-preset-env")({
      importFrom: ["@jobber/foundation"]
    })
  ]
};
```
