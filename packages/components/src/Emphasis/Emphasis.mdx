# Emphasis

<ComponentStatus stage="ready" responsive="yes" accessible="yes" />

Use this component to emphasize a certain content of a Heading or Text
component. Used to call attention to a text

```ts
import { Emphasis } from "@jobber/components/Emphasis";
```

<Playground>
  <Text>
    To <Emphasis variation="bold">boldly</Emphasis> goâ€¦
  </Text>
</Playground>

## Props

<Props of={Emphasis} />

---

## Bold

Used to call attention to a text

<Playground>
  <Text>
    <Emphasis variation="bold">Save $240</Emphasis> when you choose our Annual
    Plan
  </Text>
</Playground>

## Italic

Used to call attention to a text but it's less important than a bold

<Playground>
  <Text variation="subdued">
    <Emphasis variation="italic">Job note linked to related invoice</Emphasis>
  </Text>
</Playground>

## Highlight

Used to call attention to a specific value prop on a heading. Highlight should
only be used on page titles and subtitle.

<Playground>
  <Heading level={1}>
    Get paid <Emphasis variation="highlight">faster</Emphasis>
  </Heading>
</Playground>
