# Countdown

```ts
import { Countdown } from "@jobber/components/Countdown";
```
Countdown is used to ensure that a "countdown" to a specified end time is displayed in a consistent format. No text styling is applied to Countdown; it is only concerned with the text content that will be presented.


<Playground>
  {() => {
    const inADayOrSo = useMemo(
      () => new Date(new Date().getTime() + 25 * 3600 * 1000),
      []
    );
    return (
      <Countdown date={inADayOrSo} granularity={"dhms"} showUnits={true} />
    );
  }}
</Playground>

## On Complete

<Playground>
  {() => {
    const [start, setStart] = useState(false);
    let inTenSeconds = useMemo(
      () => new Date(new Date().getTime() + 10 * 1000),
      [start]
    );
    return (
      <>
        <Button label={"Start Timer"} onClick={() => setStart(true)} />
        <div />
        <Heading variation={"highlight"}>
          {start ? (
            <Countdown
              date={inTenSeconds}
              granularity={"s"}
              showUnits={true}
              onComplete={() => alert("Completed Timer!")}
            />
          ) : (
            ""
          )}
        </Heading>
      </>
    );
  }}
</Playground>

## Styling

Countdown is un-opinionated about its' styling. You can wrap a Countdown in [Heading](/packages-components-src-heading-heading) or [Text](/packages-components-src-text-text) to achieve the desired typographic treatment for your use case.

<Playground>
  {() => {
    const inTenHours = useMemo(
      () => new Date(new Date().getTime() + 36000 * 1000),
      []
    );
    return (
      <Content>
        <Text>
          <Countdown
            date={inTenHours}
            granularity={"ms"}
            showUnits={true}
            onComplete={() => alert("Completed Timer!")}
          />
        </Text>
        <Countdown
          date={inTenHours}
          granularity={"ms"}
          showUnits={true}
          onComplete={() => alert("Completed Timer!")}
        />
        <div />
        <Heading level={1}>
          <Emphasis variation="highlight">
            <Countdown
              date={Date.now() + 100000}
              granularity={"s"}
              showUnits={true}
              onComplete={() => alert("Completed Timer!")}
            />
          </Emphasis>
        </Heading>
      </Content>
    );
  }}
</Playground>
