# Select

<ComponentStatus stage="pre" responsive="no" accessible="no" />

In Jobber a `Select` is used to present a defined list of options for the user
to choose from.

## Simple Example

<Playground>
  <Select>
    <Option value="sad">Tony</Option>
    <Option value="old">Steve</Option>
  </Select>
</Playground>
