# Select

<ComponentStatus stage="ready" responsive="yes" accessible="yes" />

In Jobber a `Select` is used to present a defined list of options for the user
to choose from.

```ts
import { Select, Option } from "@jobber/components/Select";
```

<Playground>
  <Select>
    <Option value="tony">Tony</Option>
    <Option value="quincy">Quincy</Option>
    <Option value="peppa">Peppa Pig</Option>
  </Select>
</Playground>
