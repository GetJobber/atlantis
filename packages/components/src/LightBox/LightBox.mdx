# Light Box

<ComponentStatus stage="pre" responsive="no" accessible="no" />

LightBox is a component designed to display an array of images to the user. A
user clicks on a control and is shown a carousel style presentation of the
defined images. This component is a wrapper of the frontend-collective's
react-image-lightbox third party library.

```ts
import { LightBox } from "@jobber/components/LightBox";
```

## Design & Usage Guidelines

A lightbox should:

- Resemble the industry standard lightbox, using Jobber's font and color
  palette. Novel lightbox functionality will surprise and confuse users.

## Images

Images is an array of objects defining a LightBox image. This object consists of
`title`, `caption` and `url`. `title` and `caption` are optional, `url` is
required, for each image.
