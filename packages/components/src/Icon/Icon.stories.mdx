# Icon

Atlantis provides built-in icons that represent common tasks and types of
content in a variety of use cases.

Icons are used to visually communicate core parts of the product and available
actions. They can act as wayfinding tools to more easily understand where they
are in the product, and common interaction patterns that are available.

Use icons as intended. Provided icons have a specific meaning. To avoid
confusing users, each icon should be used in accordance with its meaning and
recommended usage.

## Design guidelines

### Icon sizes

#### Base

Use the `base` size for most icon usage.

#### Small

Only use `small` when space is severely constrained, such as in a
[small Button.](../?path=/docs/components-button--button#sizes)

#### Large

Only use `large` icons when the icon is a prominent feature of the interface,
such as in a full-screen error view on a mobile device.

### Icon construction guidelines

When contributing a new icon, design to these guidelines for the `base` size.
The `small` and `large` sizes will be generated programatically once the `base`
size icon has been created.

#### Size, weight, and spacing

Icons are only responsible for their own spacing. Spacing when used within other
components, such as [List](../?path=/docs/components-list--list), is the
responsibility of those other components.

- Icons should have a height and width of 20px container within a 24px artboard,
  effectively providing a 2px padding on all sides
- Icons can be optically adjusted within this 20px space to attain good visual
  balance with the rest of the Atlantis icon set
- Circular icons should have a 20px diameter
- Lines should have a 2px weight

#### Corners

- 90° corners should have a 2px radius
- Greater than 90° corners should have a 1px radius
- Less than 90° corners should have an 0.5px radius
- Interior corners should not have a radius

#### Line terminals

- When a line is interrupted, terminals should be geometric/square
- when a line ends organically, terminals should be round

#### Color

Design your icons with `#000000`. If an icon is associated with a systemic color
such as ["destructive"](../?path=/docs/design-colors--page#destructive) specify
the system color for the icon when it is added to Atlantis. In all other cases,
use the default icon color.
