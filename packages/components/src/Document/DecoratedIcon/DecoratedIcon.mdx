# Decorated Icon

Decorated icons are used to wrap an icon in a shape. The color of the shape is inherited from the `accentColor` of the `<Document />`.

## Basic Usage

<Playground>
  <DecoratedIcon iconName={IconName.invoice} />
</Playground>
