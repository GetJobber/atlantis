# Input Text

<ComponentStatus stage="rc" responsive="yes" accessible="yes" />

Input text is used in forms that accept short or long answers from users.

## Short answer

Use this to allow users to provide short answers.

<Playground>
  {() => {
    const [value, setValue] = useState("Veintisiete");
    const handleChange = newValue => {
      setValue(newValue);
    };
    return (
      <InputText
        value={value}
        onChange={handleChange}
        name="age"
        placeholder="Age in words"
      />
    );
  }}
</Playground>

## Long answer

Use this to allow users to provide long answers.

<Playground>

## Use Refs

Use this to allow users to provide short answers.

<Playground>
  {() => {
    const ref = React.createRef();
    const focusInput = () => {
      ref.current.focus();
    };
    return (
      <>
        <InputText name="age" ref={ref} placeholder="Age in words" />
        <input
          type="button"
          value="Focus the text input"
          onClick={focusInput}
        />
      </>
    );
  }}
</Playground>
