# Avatar

<ComponentStatus stage="pre" responsive="no" accessible="yes" />

An avatar will be used to display an visual identifier for an individual user.

```ts
import { Avatar } from "@jobber/components/Avatar";
```

    name="The Jobbler"
    size="large"
    initials="JBLR"
    color="var(--color-white)"
  />
</Playground>

## Color

Use `color` to indicate the background and the border color of an avatar
component.

> There is currently a deficiency in color when handling color contrast. It is
> currently recommended not use a light color for the color prop.

<Playground>
