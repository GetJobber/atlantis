# Content

<ComponentStatus stage="ready" responsive="yes" accessible="yes" />

Content is a container component which provides its' child elements with
consistent vertical spacing and padding.

```ts
import { Content } from "@jobber/components/Content";
```

## Design & usage guidelines

Content should be used whenever you have a container with mixed content that
requires consistent spacing between the elements.
