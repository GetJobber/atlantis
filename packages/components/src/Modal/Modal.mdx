# Modal

<ComponentStatus stage="pre" responsive="no" accessible="no" />

Modals are overlays that allow users to view, edit, or show informations that
doesn't require a page to be built. It also prevent users from interacting with
the rest of the application until a specific action is taken.

<Playground>
  {() => {
    const [value, setValue] = useState(false);
    return (
      <>
        <Modal title="Bob" open={value} onRequestClose={() => setValue(!value)}>
          Haloo
        </Modal>

        <Button
          label="Open Modal"
          onClick={() => setValue(!value)}
        />
      </>
    );

}}

</Playground>

## With actions

You have 3 action props you can use

- `primaryAction`
- `secondaryAction`
- `tertiaryAction`

<Playground>
  {() => {
    const [value, setValue] = useState(true);
    const toggleModal = () => {
      setValue(!value);
    };
    const handlePrimaryAction = () => {
      alert("✅");
    };
    const handleSecondaryAction = () => {
      toggleModal();
    };
    const handleTertiaryAction = () => {
      alert("❌");
    };
    return (
      <>
        <Modal
          title="Bob"
          open={value}
          primaryAction={{ label: "Submit", onClick: handlePrimaryAction, fullWidth: true }}
          secondaryAction={{ label: "Cancel", onClick: handleSecondaryAction, fullWidth: true }}
          tertiaryAction={{ label: "Delete", onClick: handleTertiaryAction }}
          onRequestClose={toggleModal}
        >
          Haloo
        </Modal>

        <Button
          label="Open Modal"
          onClick={toggleModal}
        />
      </>
    );

}}

</Playground>
