# Modal

Modals are overlays that allow users to view, edit, or show informations that
doesn't require a page to be built. It also places the user in a _mode_ of
interaction where the rest of the application is unavailable until the Modal
action is completed or canceled, hence the name Modal.

```ts
import { Modal } from "@jobber/components/Modal";
```

## With actions

You have 3 action props you can use

- `primaryAction`
- `secondaryAction`
- `tertiaryAction`
