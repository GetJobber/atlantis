# TextField

<ComponentStatus stage="rc" responsive="yes" accessible="yes" />

A text field is used in forms that accept short or long answers from users.

## Short answer

Use this to allow users to provide short answers.

<Playground>
  <TextField placeholder="Company name" />
</Playground>

## Long answer

Use this to allow users to provide long answers.

<Playground>
  <TextField placeholder="Describe what your company does" multiline={true} />
</Playground>

## States

### Disabled

<Playground>
  <TextField
    placeholder="Credit card"
    value="**** **** **** 1234"
    disabled={true}
    name="creditCard"
  />
</Playground>

### Read only

<Playground>
  <TextField
    placeholder="Your referral code"
    value="WXYZ100"
    readonly={true}
    name="referralCode"
  />
</Playground>

### Invalid

<Playground>
  <TextField
    placeholder="Email"
    value="atlantis"
    invalid={true}
    name="emailAddress"
  />
</Playground>
