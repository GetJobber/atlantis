## Alignment

For text alignment, we use `left` and `right`, but the CSS values use `start`
and `end` internally. This allows us to support
[bi-directionality](https://material.io/design/usability/bidirectionality.html#mirroring-layout)
when needed.

<Playground>
  <Typography align="left">Left</Typography>
  <Typography align="center">Center</Typography>
  <Typography align="right">Right</Typography>
</Playground>
