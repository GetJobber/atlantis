  {() => {
    const checkboxRef = useRef();
    const [showPopover, setShowPopover] = useState(true);

    const onChangeSubscribed = () => {

            label="Subscribe on marketing emails"

            onChange={onChangeSubscribed}
          />
        </span>
        <div>
          <Button
            label="Restart Scenario"
            onClick={() => setShowPopover(true)}
          />
        </div>
        <Popover
          attachTo={checkboxRef}
          open={showPopover}
          dismissible={true}
          preferredPlacement="right"
          title="Show popover on load!"
          onRequestClose={() => setShowPopover(false)}
        >
          <Content>Some amazing content</Content>
          <Content>
            <Button
              label="Disappear forever"
              onClick={() => setShowPopover(false)}
            />
          </Content>
        </Popover>
      </>
    );
  }}

## Design & Usage Guidelines

When contributing to, or consuming the Popover component, consider the
following:

### Usage

- A popover should be used to introduce a novel UI element or highlight a change
  to an existing interface.
- A popover should contain information that is helpful, but not necessary for
  the user to successfully interact with the page.
- Once a popover has been shown, it should not show up again once the user has
  interacted with it.

### Content

- Popover text content should not exceed 3 lines and can consist of an optional
  "dismiss" CTA

### Reference Docs

- Documentation on the popper plugin that was used to build the popover
  component https://popper.js.org/react-popper/v2/
