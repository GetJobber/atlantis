## Design & Usage Guidelines

When contributing to, or consuming the ConfirmationModal component, consider the
following:

- When using the ConfirmationModal component always consider the volume you
  would like it to be displayed at.
- The ConfirmationModal component should never be used inside another component.

## Controlled Confirmation Modal

In some instances, such as a list, you may want to place a single Confirmation
Modal on a page and then call to open it when required.

<Playground>
  {() => {
    const confirmationModalRef = useRef();
    const users = [
      {
        id: 1,
        name: "Bob"
      },
      {
        id: 2,
        name: "Donald"
      }
    ];
    return (
      <>
        {users.map(user => {
          return (
            <Button
              label={`Confirm ${user.name}`}
              onClick={() => {
                confirmationModalRef.current.show({
                  title: "Should we?",
                  text: `Hang out with ${user.name}?`,
                  confirmLabel: "Hangout",
                  onConfirm: () => {
                    alert("✅");
                  }
                });
              }}
            />
          );
        })}
        <ConfirmationModal ref={confirmationModalRef} />
      </>
    );
  }}
</Playground>
