# Text

<ComponentStatus stage="pre" responsive="no" accessible="yes" />

Text is used to add an additional visual meaning on your content

## Default

Use this as the body text

<Playground>
  <Text>Ask the information you need upfront from clients and new leads</Text>
</Playground>

## Subdued

De-emphasize a piece of text that is less important to the user

<Playground>
  <Text variation="subdued">Job note linked to related invoice</Text>
</Playground>

## Intro

Used as an intro text that gives more infor about the page title. This can only
be used right after a page title.

<Playground>
  <Text variation="intro">
    Attract new customers by asking happy clients for a Facebook recommendation
  </Text>
</Playground>

## Feedback message

<Playground>
  <Text variation="success">Invoice sent</Text>
  <br />
  <Text variation="error">Name is required</Text>
  <br />
  <Text variation="warning">Your message is over 160 characters</Text>
  <br />
  <Text variation="info">
    Drag to rearrange the order that fields show up in Jobber
  </Text>
</Playground>

## Button

The `button` variation should only be used with a `Button` component

<Playground>
  <Text variation="button">Update Quote</Text>
</Playground>
