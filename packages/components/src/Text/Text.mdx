## Default

Use this as the body text

<Playground>
  <Text>Ask the information you need upfront from clients and new leads</Text>
</Playground>

## Subdued

De-emphasize a piece of text that is less important to the user

<Playground>
  <Text variation="subdued">Job note linked to related invoice</Text>
</Playground>

## Intro

Used as an intro text that gives more information about the page title. This can
only be used right after a page title.

<Playground>
  <Text variation="intro">
    Attract new customers by asking happy clients for a Facebook recommendation
  </Text>
</Playground>

total number of lines does not exceed this number.
