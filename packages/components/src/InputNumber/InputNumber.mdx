# InputNumber

<ComponentStatus stage="pre" responsive="yes" accessible="yes" />

Input number is used in forms that accept numbers as an answer.

<Playground>
  {() => {
    const [value, setValue] = useState(30);
    return (
      <InputNumber
        value={value}
        max={32}
        min={0}
        onChange={setValue}
        placeholder="Age"
      />
    );
  }}
</Playground>
