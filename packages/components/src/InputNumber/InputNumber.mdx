# InputNumber

<ComponentStatus stage="pre" responsive="no" accessible="no" />

Input number is used in forms that accept numbers as an answer.

## Simple Example

<Playground>
  {() => {
    const [value, setValue] = useState(1234567890);
    const handleChange = newValue => {
      setValue(newValue);
    };
    return (
      <InputNumber value={value} onChange={handleChange} placeholder="Age" />
    );
  }}
</Playground>
