# Atlantis overview

Atlantis is Jobber’s design system. It holds the foundational design tokens
shared across our web and mobile platforms, as well as a library of re-usable
React components for the web and mobile, and guideance for patterns and
practices for building with them.

Atlantis is maintained by the
[UX Foundations](https://jobber.atlassian.net/wiki/spaces/TEAMATL) team, but
welcomes contributions.

## Using Atlantis in your work

You can leverage Atlantis everywhere from ideation to development. Learn more
about how Atlantis fits into the larger structure of our product, from design
tokens through to fully-built features, and how to evolve Atlantis by
contributing your ideas and insights back into the system.
