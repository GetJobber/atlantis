# Grammar and structure

## Explaining prerequisites

If the user has to perform one action before they can perform another, lead with
the verb in any instructions so that they understand what _kind_ of action is
required.

| ✅ **Do **                                         | ❌ **Don't**                                                |
| -------------------------------------------------- | ----------------------------------------------------------- |
| Turn on **show pricing** to give access to reports | **Show pricing** must be turned on in order to view reports |
