## Row Actions (On Hover)

Row actions appear when hovering over individual rows:

```tsx
const [hoveredRow, setHoveredRow] = useState<string | null>(null);

<DataTable.Row
  onMouseEnter={() => setHoveredRow(row.id)}
  onMouseLeave={() => setHoveredRow(null)}
>
  <DataTable.Cell>Content</DataTable.Cell>

  {hoveredRow === row.id && (
    <DataTable.RowActions>
      <Button
        icon="checkmark"
        ariaLabel="Complete"
        type="secondary"
        variation="subtle"
      />
      <Button
        icon="edit"
        ariaLabel="Edit"
        type="secondary"
        variation="subtle"
      />
    </DataTable.RowActions>
  )}
</DataTable.Row>;
```
