# Select

A Select is used to present a defined list of options to choose from.

## Design & usage guidelines

Nested within the Select component, Option defines the options that can be
selected. For grouping options with section headers, use Select.OptionGroup.

#### Browser support

The grouped menu styling (bold section headers, edge‑to‑edge dividers and hover backgrounds) is enhanced when the browser supports `appearance: base-select` (Chromium 123+). In unsupported browsers the Select gracefully falls back to the native menu while still rendering the same `OptionGroup` structure.

## Option Grouping

Use `Select.OptionGroup` to organize options with section headers for better user experience and visual hierarchy. The `label` prop is required as it provides the section header text.
