# Select

A Select is used to present a defined list of options to choose from.

## Design & usage guidelines

Within the Select component, use Option to define the options that can be
selected.

#### Option Props

<ArgsTable of={Option} />

To set the default value of the Select, use the `value` prop on the Select
component. ////////
