# FormatTime

In Jobber a FormatTime is used to ensure that time is displayed in the expected
format. No text styling is applied, this simply formats the text.

## Specify format

You can select a specific format to use in FormatTime. See the examples below:

```tsx
const newDate = new Date();
newDate.setHours(22, 35, 0, 0);

<FormatTime time={newDate} use24HourClock={true} />

<FormatTime time={newDate} use24HourClock={true} />

<FormatTime time={newDate} use24HourClock={false} />

<FormatTime time={newDate} use24HourClock={false} />
```
